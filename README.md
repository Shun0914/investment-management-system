```bash

```

#### **2. Install Dependencies**

#### **3. Configure Claude Desktop**
Add to your Claude Desktop configuration:
```json

    "Investment Management System": {
      "command": "python",
      "args": ["server.py"],
      "cwd": "/path/to/investment-management-system"
    }
  }
}
```

